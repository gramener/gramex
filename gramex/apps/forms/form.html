Embed form

You can use any of the methods below to render the form. Note the file extensions in each of the approaches below.

{% set _path = handler.request.uri.split('/form/')[0] %} {% set _url = handler.request.protocol + '://' + handler.request.host + _path + "/embed/" + form_id %}

Bootstrap

Before anything, import Bootstrap 4 as a dependency. 
          <script src="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/4.5.3/css/bootstrap.min.css"></script>

Using JavaScript

Below code renders the form in place. 

          <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/4.5.3/css/bootstrap.min.css">
            <div class="form-container">
              <script src="{% raw _url + '.js' %}"></script>
            </div>

Using HTML

Below code renders the form in .form-container element. 

          <div class="form-container"></div>
            <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
            <script>
              $('.form-container').load("{% raw _url + '.html' %}")
            </script>

Server-side rendering

Below snippet is a Gramex server-side rendering example which renders the form in place. 

          {%! import requests %}
            {%! raw requests.get('{{ _url + '.html' }}').text %}
