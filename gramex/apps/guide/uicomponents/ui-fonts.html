Custom Fonts

You can set change the fonts using these Bootstrap variables:

  • ?font-family-base=roboto
  • ?headings-font-family=roboto-condensed
  • Reset fonts

You can pick the font value from the pre-defined fonts defined here:

    {% for font_name, font_conf in ui_config.fonts.items() %}
  • {{ font_name }}: {{ font_conf.stack }} {% if 'google' in font_conf %} (Google) {% end %}
    • {% end %}

You can also pick your font stack directly:

  • ?font-family-base=Georgia,Times,"Times New Roman",serif
  • ?font-family-base=Tahoma,Verdana,Segoe,sans-serif
  • ?heading-font-family="Gill Sans","Gill Sans MT",Calibri,sans-serif
  • Reset fonts

To use a custom font, import it before your import bootstraptheme.css. For example:

{% apply only_source %} {% end %} {% apply only_source %} {% end %}
